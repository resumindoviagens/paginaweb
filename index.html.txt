## O que fazer agora

1. Abra o repositório do website.
2. Apague o `index.html` antigo.
3. Crie um novo `index.html`.
4. Cole integralmente este código.
5. Faça:

   * Commit to main
   * Push origin
6. A Vercel atualizará automaticamente.
